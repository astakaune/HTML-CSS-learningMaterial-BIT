git clone <iterpti nuoroda> - nusiklonuoja nutolusia repozitorija. GITHUB -> Jusu PC
git add * - visus pakeitimus prideda prie commito
git add index.html - prideda viena faila pavadinimu index.html
git commit - sukelia pakeitimus i lokalia repozitorija(jusu nuklonuotas folderis)
git push - sukelia visus commitus i nutolusia repozitorija(GITHUB, GITLAB, BITBUCKET)
git pull - parsitraukia visus pakeitimus is nutolusios repozitorijos

git merge <branch name> - prijungia nurodyta brancha prie dabartinio kuriame esate
git checkout <branch name> - pakeicia i jusu nurodyta brancha
git checkout -b <branch name> - pakeicia i jusu nurodyta brancha ir ji sukuria jei neegzistuoja

1. branch feature-new-page
2. idedi savo pakeitimus(commitus)
3. Nori apjungti feature-new-page su main
4. Sukuri Pull request, kad mergini feature-new-page i main
